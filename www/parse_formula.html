Simple Parsing of Formula

Usage

parse_formula(x)

Arguments

x
formula to parse

Value

a list with the following elements: response logical that indicates if the formula has a response term. y name of the response variable. x list of regressor variable names. n number of regressor variables.

Description

Formula parser for formula-based NMF models.
