Custom Access to Viewport Paths

Usage

current.vpPath2()

Description

This function was defined to substitute current.vpPath because this latter calls grid.Call("L_currentViewport") which calls 'L_gridDirty' before effectively calling 'L_currentViewport'. This starts a new page (not sure why), which is not ok when writing to a device that records each page (e.g. PDF or SVG)
