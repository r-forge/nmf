Generic Strategy Class

Usage

name(object, ...)

S4 (Strategy)
`name`(object, all = FALSE)

name(object, ...) <- value

Arguments

object
an R object with a defined name method
...
extra arguments to allow extension
value
replacement value
all
a logical that indicates if all the names associated with a strategy should be return (TRUE), or only the first (primary) one (FALSE).

Description

This class defines a common interface for generic algorithm strategies (e.g., NMFStrategy-class).

name and name<- gets and sets the name associated with an object. In the case of Strategy objects it is the the name of the algorithm.

Slots

namecharacter string giving the name of the algorithm

packagename of the package that defined the strategy.

defaultsdefault values for some of the algorithm's arguments.

Methods

namesignature(object = "Strategy"): Returns the name of an algorithm

namesignature(object = "Strategy"): Returns the name of an algorithm

name<-signature(object = "Strategy", value = "character"): Sets the name(s) of an NMF algorithm

name<-signature(object = "Strategy", value = "character"): Sets the name(s) of an NMF algorithm
