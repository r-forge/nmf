Get Stopping Criterion

Usage

NMFStop(val)

Arguments

val
access key that can be a character string, a single integer or numeric, or a function.

Value

a function

Description

Convert a key into a stopping criterion (i.e. a function).

Details

Functions are returned unchanged, integer values are used to create a stopping criterion of that number of iterations, numeric values are used to create a stopping criterion of that stationary threshold.
