Get/Set a Static Variable in NMF Algorithms

Description

Description,c("This function is used in iterative NMF algorithms to\n manage variables stored in a local workspace, that are\n accessible to all functions that define the iterative\n schema described in\n NMFStrategyIterative-class.", "It is specially useful for computing stopping criteria,\n which often require model data from different iterations.")

Usage

staticVar(name, value, init = FALSE)

Arguments

name
Name of the static variable (as a single character string)
value
New value of the static variable
init
a logical used when a value is provided, that specifies if the variable should be set to the new value only if it does not exist yet (init=TRUE).

Value

The value of the static variable
