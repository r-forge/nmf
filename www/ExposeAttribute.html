Exposing Object Attributes

Usage

ExposeAttribute(object, mode = "rw")

EAmode(x)

EAmode(x) <- value

Arguments

object
any R object whose attributes need to be exposed
mode
access mode: “r”: (read-only) only method $ is defined “w”: (write-only) only method $<- is defined “rw”: (read-write) both methods $ and $<- are defined
x
an ExposeAttribute object
value
replacement value for mode.

Description

The function ExposeAttribute creates an S3 object that exposes all attributes of any R object, by making them accessible via methods $ and/or $<-.

EAmode and EAmode<- get and sets the access mode of ExposeAttribute objects.
